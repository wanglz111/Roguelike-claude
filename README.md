# Roguelike AI Dev Project

一个为 AI 持续接力开发而设计的 CLI Roguelike / 魔塔式 RPG 项目。

游戏特性：

1. 支持中英文双语
2. CLI 命令行界面
3. 回合制战斗系统
4. 楼层探索与成长机制

当前版本状态：

**✅ Milestone 1-3 已全部完成** - 游戏完全可玩！

核心功能：
1. 完整的战斗系统（回合制、暴击、装备加成、技能、状态效果）
2. 丰富的内容池（30 种怪物：26 普通 + 4 Boss，覆盖 1-20 层）
3. 成长系统（经验、升级、属性提升）
4. 装备系统（武器、护甲、饰品、属性加成、稀有度，30 件装备）
5. 装备套装系统（5 个套装，穿戴整套获得额外加成）
6. 背包系统（物品收集、使用、装备）
7. 技能系统（30 种主动技能：7 种通用 + 20 种职业专属 + 3 种带状态效果，消耗 MP）
8. 职业系统（5 种职业：战士、法师、盗贼、游侠、牧师，不同基础属性、成长速率和专属技能）
9. 状态效果系统（6 种状态效果：中毒、燃烧、冰冻、流血、再生、虚弱，影响战斗策略）
10. 事件系统（30 个随机事件，多种效果类型，包括永久属性提升）
11. 消耗品系统（9 种消耗品：3 种生命药水 + 3 种魔力药水 + 3 种特殊药剂，包括状态治疗和紧急恢复）
12. 商店系统（5 种商店类型，购买物品和装备，包括 MP 恢复药水）
13. 掉落系统（基于楼层的智能掉落表，Boss 掉落更好的装备，稀有度随楼层提升）
14. 存档系统（保存/加载完整游戏状态，支持 3 个独立存档槽位）
15. Boss 系统（每 5 层出现，特殊挑战）
16. 周目系统（New Game+，通关后可开始新周目，保留进度，难度递增）
17. 成就系统（43 种成就，追踪玩家里程碑和特殊成就，包括难度相关成就）
18. 难度选择（简单/普通/困难，影响怪物属性、金币和掉落率）
19. 数值平衡（已测试并调整）
20. 双语支持（中英文）
21. 彩色 UI（自动检测终端支持，语义化颜色提升可读性）

游戏可以从第 1 层玩到第 20 层，体验完整的 Roguelike 游戏循环。

## 运行方式

```bash
python3 -m cli.main
```

游戏启动时会提示选择语言（中文或英文）。

游戏支持 3 个独立的存档槽位，可以在不同槽位保存多个游戏进度。

## Claude 自动循环

如果你已经配置好 `claude` CLI，可以让它自动连续跑多轮开发：

```bash
bash scripts/claude_auto_loop.sh 5
```

含义：

1. 每轮都会先读取 `ai_dev/` 规则文件
2. 每轮只完成 `next_tasks.md` 中的一个小任务
3. 每轮结束后自动进入下一轮
4. Claude 的输出会保存到 `ai_dev/logs/`
5. 如果仓库有改动，脚本会自动提交当前这一轮的变更

常用环境变量：

```bash
CLAUDE_MODEL=sonnet
CLAUDE_EFFORT=medium
CLAUDE_PERMISSION_MODE=default
CLAUDE_AUTO_COMMIT=1
```

例如：

```bash
CLAUDE_MODEL=sonnet CLAUDE_EFFORT=high bash scripts/claude_auto_loop.sh 10
```

停止方式：

```bash
touch ai_dev/STOP
```

脚本会在当前轮结束后停止。
