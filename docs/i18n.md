# 双语支持文档 / Bilingual Support Documentation

## 概述 / Overview

本项目支持中英文双语。所有面向玩家的文本内容都提供了中英文版本。

This project supports both Chinese and English. All player-facing text content is available in both languages.

## 使用方法 / Usage

### 启动游戏 / Starting the Game

运行游戏时，会首先提示选择语言：

When starting the game, you will be prompted to select a language:

```bash
python3 -m cli.main
```

输入 `en` 选择英文，输入 `zh` 选择中文。

Enter `en` for English, `zh` for Chinese.

### 演示脚本 / Demo Script

运行演示脚本查看双语效果：

Run the demo script to see bilingual support in action:

```bash
python3 demo_i18n.py
```

## 开发指南 / Development Guide

### 数据格式 / Data Format

所有内容数据（怪物、物品、事件等）都应使用以下格式：

All content data (monsters, items, events, etc.) should use the following format:

```json
{
  "name": {
    "en": "English Name",
    "zh": "中文名称"
  },
  "description": {
    "en": "English description",
    "zh": "中文描述"
  }
}
```

### 代码中使用 / Using in Code

在代码中使用 `t()` 函数进行翻译：

Use the `t()` function in code for translation:

```python
from game.i18n import t

# 简单文本 / Simple text
message = t({"en": "Hello", "zh": "你好"})

# 带变量的文本 / Text with variables
player_name = "Hero"
message = t({"en": f"Welcome, {player_name}.", "zh": f"欢迎，{player_name}。"})
```

### 切换语言 / Switching Language

```python
from game.i18n import get_i18n

# 设置为英文 / Set to English
get_i18n().set_language("en")

# 设置为中文 / Set to Chinese
get_i18n().set_language("zh")
```

### 实体类方法 / Entity Class Methods

`Monster` 和 `Item` 类提供了便捷方法：

`Monster` and `Item` classes provide convenience methods:

```python
monster = Monster(name={"en": "Goblin", "zh": "哥布林"}, ...)
print(monster.get_name())  # 根据当前语言返回名称 / Returns name based on current language

item = Item(name={"en": "Potion", "zh": "药水"}, ...)
print(item.get_name())  # 根据当前语言返回名称 / Returns name based on current language
print(item.get_description())  # 根据当前语言返回描述 / Returns description based on current language
```

## 添加新内容 / Adding New Content

### 怪物 / Monsters

编辑 `content/monsters.json`：

Edit `content/monsters.json`:

```json
{
  "name": {
    "en": "New Monster",
    "zh": "新怪物"
  },
  "min_floor": 1,
  "hp": 20,
  "attack": 8,
  "defense": 2,
  "exp_reward": 10,
  "gold_reward": 5,
  "drop_item": "Small Potion"
}
```

### 物品 / Items

编辑 `content/items.json`：

Edit `content/items.json`:

```json
{
  "name": {
    "en": "New Item",
    "zh": "新物品"
  },
  "item_type": "consumable",
  "effect_type": "heal",
  "effect_value": 20,
  "description": {
    "en": "Restores 20 HP",
    "zh": "恢复20点生命值"
  }
}
```

## 测试 / Testing

所有现有测试都已通过，双语支持不影响游戏逻辑。

All existing tests pass. Bilingual support does not affect game logic.

```bash
python3 -m unittest discover -s tests -p "test_*.py" -v
```

## 注意事项 / Notes

1. 所有新增内容必须提供中英文双语版本
2. 使用 `t()` 函数而不是硬编码字符串
3. 保持两种语言的文本长度和风格一致
4. 数字、符号等不需要翻译的内容可以直接使用

---

1. All new content must provide both Chinese and English versions
2. Use the `t()` function instead of hardcoded strings
3. Keep text length and style consistent between languages
4. Numbers, symbols, and other non-translatable content can be used directly
